SF Bay Area. I have an M.A. in Interaction Design and a habit of not stopping at the design file.

## ⚡ How I Build

I'm AI-native — not AI-assisted. My actual workflow:

```
Claude Code (agentic) → SwiftUI → Gemini AI API → RevenueCat → TestFlight → App Store
```

- **Agentic pipeline:** Mac Mini server running Claude Code headlessly, orchestrating tasks overnight
- **Design system:** Figma components → SwiftUI views, minimal drift between spec and ship
- **Monetization-first:** RevenueCat wired from day one — subscriptions, trials, paywalls
- **ASO as product:** Metadata, screenshots, and keywords are part of the build, not an afterthought
